# Bilingual Sync Policy
Chinese repo: https://github.com/yw901121-creator/ai-learn
English repo: https://github.com/yw901121-creator/ai-learn-english-

Whenever course content is added or updated, BOTH repos must be updated in the same session:
1. Update Chinese files in ai-learn / local ai-course
2. Translate/update matching English files here
3. Push both repos
4. Keep folder structure and filenames identical across languages

Trigger phrase for the maintainer agent: "更新 AI 課程新聞" or "update AI course news" should refresh BOTH language versions.
