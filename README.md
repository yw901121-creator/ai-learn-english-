## What does this course teach?

You will learn:

1. **The AI fundamentals map** — from ML -> Deep Learning -> Generative AI
2. **Core LLM concepts** — Prompt, Token, Context, RAG, Fine-tuning
3. **Hot technologies in 2026** — Agent, MCP, Multi-Agent, Computer Use
4. **EdTech AI topics** — Adaptive Learning, Knowledge Tracing, AI Tutor
5. **Advanced Agent x Context topics** — Agent Loop, Harness, Skills, Context Compact, Lost in the Middle, OCR+RAG
6. **Interview talk tracks** — complete 30-second and 2-minute answers + follow-up handling
7. **Daily news updates** — new models, new terminology, industry movement

---

## Recommended learning path (8-day sprint)

| Day | What to read | Goal |
|---|---|---|
| Day 1 | `00-overview` + `01-fundamentals` | Be able to draw the AI map |
| Day 2 | `02-llm-deep-dive` | Be able to explain LLM / RAG / Fine-tuning |
| Day 3 | `03-agents-tools` | Be able to explain Agent / MCP / Tool Use |
| Day 4 | `04-edtech-ai` | Be able to talk through five EdTech AI use cases |
| Day 5 | `08-agent-context` | Be able to explain Loop / Harness / Context Engineering |
| Day 6 | `05-glossary` | Memorize 60+ key terms |
| Day 7 | `06-interview` | Run mock interviews and rehearse answers |
| Day 8 | `07-news-updates` | Stay current and earn bonus points |

Plan for about **45-90 minutes** per day. Finish at least Day 1-7 before your interview.

## Daily update workflow

I **cannot run automatically in the background every day** unless you set up Cursor Automation or come back and ask me each day.

**The simplest approach: send me this once a day**

```
Update the AI course news
```

Then I will:
1. Search for the latest AI news
2. Update `07-news-updates/LATEST.md`
3. If there are new terms -> add them to `05-glossary`
4. If there are major changes -> revise the related lessons and alert you

See [`HOW-TO-DAILY-UPDATE.md`](HOW-TO-DAILY-UPDATE.md) for details.
