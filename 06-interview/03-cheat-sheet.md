# Pocket Cheat Sheet (10 Minutes Before the Interview)

## 30-second opener
AI x Education = personalization + automation + data insight; LLM + RAG + Agent; AI accelerates teachers rather than replacing them.

## Must-memorize reference
| Term | One-line explanation |
|---|---|
| LLM | Large Language Model; generates text/code |
| RAG | Retrieve course materials first, then answer |
| OCR | Extract text from images/scans |
| Agent | Plans and calls tools to get work done |
| Agent Loop | Perceive -> Plan -> Act -> Observe -> Reflect |
| Harness | The execution framework around the model (tools / permissions / memory) |
| Skills | Reusable, on-demand professional capability packages |
| Context Compact | Compress old context to make room for new information |
| Lost in the Middle | Middle portions of long context are easy to ignore |
| MCP | A standard connector for AI-to-tool integration |
| Hallucination | Fabricated output; especially dangerous when teaching |
| Knowledge Tracing | Tracking mastery of knowledge points |
| Adaptive Learning | Dynamically adjusts content based on learner level |
| Guardrails | Safety controls and permission boundaries |

## How to make AI "notice the important parts" in context
1. Pin the rules  
2. Keep a state card (`goal / constraints / progress`)  
3. In RAG, include only the top refined evidence  
4. Put key instructions at the **beginning + end**  
5. Compact old turns instead of dumping all 20 turns in full

## Five major applications
Teaching assistants, adaptive learning, automated assessment, content generation, learning alerts

## The four risk buckets
Teaching errors / cheating / bias / privacy

## Strong one-liners
1. AI makes one-on-one learning scalable  
2. Guidance beats giving away the answer  
3. AI can be on in practice, but must be off in exams  
4. High-stakes outcomes need Human-in-the-loop  
5. Choose models by accuracy, cost, latency, and controllability  
6. Fitting into context does not mean being used well (Lost in the Middle)  
7. The model is the brain; the Harness is the body that can actually go to production
